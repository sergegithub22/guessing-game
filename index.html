description

To play this game you must turn on JavaScript option in your browser

In this game you have to guess a number which was given randomly. Range from 0 to 10 inclusively.

How to play:
Enter a number from 0 to 10 inclusively by screen keyboard, then press button "Compare". Result you will see in field "Result".

If you guess - collect the score equal 4. For miss - minus 1 score. You begin with 4 scores

In this game present a counters: round attempts, total attempts, scores and percents guessing.
