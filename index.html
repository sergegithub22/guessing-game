number guessing game

description

To play this game you must turn on JavaScript option in your browser

In this game you have to guess a number which was given randomly. Range from 0 to 10 inclusively.

Game interface comprises (from top to below): Header "Game", header "Info", information field, advancer information (attempts left, guesses), entry field, screen keyboard

How to play:
Enter a number from 0 to 10 inclusively by screen keyboard, then press button "Compare". Result you will see in information field. For clear entry field press button "Del"

You have a 30 attempts to guess. For 0 attempts new game will starts

In the information field you can see current information

Also, below information field, you can see how much attempts is left and guessing percents

Themes. In section "Settings" you can set theme would you like: light or dark. Dark is default
